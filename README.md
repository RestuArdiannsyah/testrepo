# Kalkulator Bunga Sederhana (Simple Interest Calculator)

Sebuah skrip berbasis Python/Shell yang digunakan untuk menghitung bunga sederhana berdasarkan pokok pinjaman, tingkat bunga tahunan, dan jangka waktu dalam tahun.

## Deskripsi
Proyek ini bertujuan untuk memberikan alat bantu cepat bagi pengguna untuk menghitung total bunga yang terakumulasi selama periode tertentu. Rumus yang digunakan adalah:

**Bunga Sederhana = (Pokok * Tingkat Bunga * Waktu) / 100**

## Fitur
* Menghitung bunga tahunan dengan input yang mudah.
* Validasi input untuk memastikan angka yang dimasukkan benar.
* Output yang bersih dan mudah dibaca.
